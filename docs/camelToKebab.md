# camelToKebab

Converts a camelCase string to kebab-case.

## Signature
`String -> String`

## Examples
```javascript
camelToKebab('helloWorld')     // 'hello-world'
camelToKebab('HTTPRequest')    // '-h-t-t-p-request'
camelToKebab('HelloWorld')     // '-hello-world'
camelToKebab('')               // ''
```
